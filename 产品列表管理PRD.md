## 2. 本次新增/变更内容

本次在“产品模板管理”页面补齐模板列表、筛选、分页和配置广告弹窗能力。

| 新增/变更项 | 用户能看到什么 | 说明 |
| --- | --- | --- |
| 产品模板管理列表 | 页面展示模板设置页签、筛选区、模板列表和分页 | 用于管理员查看和维护产品模板。 |
| 模板筛选 | 支持按产品类型、模板名称、内容类型、创建时间筛选 | 当前原型为静态演示，真实系统需接入查询接口。 |
| 配置广告入口 | 每条模板操作列展示“配置广告”按钮 | 点击后打开配置广告弹窗。 |
| 配置广告弹窗 | 弹窗展示 APP端、WEB端广告位列表 | 支持勾选广告位、切换端类型、分页展示和确认保存。 |
| 广告搜索 | 弹窗内新增搜索框、搜索按钮、重置按钮 | 支持按广告位名称筛选当前端的广告位。 |

## 3. 背景

产品模板用于承载不同产品形态和内容类型的配置。运营或管理员需要在模板维度维护菜单、广告、打包等配置，保证不同模板在前台使用时具备对应的展示能力。

当前需要补齐“配置广告”能力，让管理员可以在模板列表中进入广告配置弹窗，并快速查找需要启用或关闭的广告位。

## 4. 产品目标

- 管理员可在产品模板列表查看模板基础信息和状态。
- 管理员可通过筛选条件快速定位模板。
- 管理员可在模板维度配置 APP端、WEB端广告位。
- 配置广告弹窗支持搜索，降低广告位较多时的查找成本。
- 弹窗保存后，系统可记录该模板的广告配置变更。

## 7. 配置广告弹窗

### 7.1 入口

入口位置：模板列表每行操作列中的 `配置广告` 按钮。

点击后打开弹窗，弹窗标题为：`配置广告`。

### 7.2 弹窗基础布局

弹窗包含：

| 区域 | 内容 |
| --- | --- |
| 标题区 | 标题“配置广告”和关闭按钮。 |
| 端类型页签 | APP端、WEB端。默认选中 APP端。 |
| 搜索区 | 广告名称输入框、搜索按钮、重置按钮。 |
| 广告位列表 | 当前端类型下的广告位复选框列表。 |
| 分页区 | 总条数、每页条数、页码。 |
| 底部操作 | 取消、确定。 |

### 7.3 端类型页签

| 页签 | 说明 |
| --- | --- |
| APP端 | 展示 APP 相关广告位。 |
| WEB端 | 展示 WEB 相关广告位。 |

切换规则：

1. 默认进入弹窗时选中 APP端。
2. 点击 WEB端后，列表切换为 WEB端广告位。
3. 切换端类型时，清空当前搜索关键词。
4. 不同端类型的广告勾选状态互相独立。

### 7.4 广告位列表

列表以复选框形式展示广告位。

广告位示例：

| 端类型 | 广告位名称 |
| --- | --- |
| APP端 | Y06APP 菜单插入广告 |
| APP端 | 16:9【A】帖子详情页底部横幅 |
| APP端 | 【A】帖子详情页顶部横幅 |
| APP端 | 16:9【A】个人中心中部横幅 |
| APP端 | 3:4【A】个人中心弹窗广告 |
| APP端 | 16:9【A】播放结束覆盖广告 |
| APP端 | 16:9【A】播放暂停覆盖广告 |
| APP端 | 16:9【A】漫画/写真详情底部横幅 |
| APP端 | 16:9【A】应用中心底部横幅 |
| APP端 | 1:1【A】搜索页顶部图标位广告-A |
| WEB端 | WEB端 首页顶部横幅 |
| WEB端 | WEB端 首页底部横幅 |
| WEB端 | WEB端 搜索结果页顶部广告 |
| WEB端 | WEB端 详情页右侧广告 |

展示规则：

1. 已配置的广告位默认勾选。
2. 未配置的广告位默认不勾选。
3. 用户可手动勾选或取消勾选广告位。
4. 广告位名称过长时可单行省略，鼠标悬停展示完整名称。

### 7.5 搜索功能

搜索范围：当前选中端类型下的广告位名称。

搜索控件：

| 控件 | 说明 |
| --- | --- |
| 输入框 | 占位文案：请输入广告名称。 |
| 搜索 | 按输入关键词筛选广告位。 |
| 重置 | 清空关键词，并恢复当前端类型下完整广告位列表。 |

搜索规则：

1. 支持广告位名称模糊搜索。
2. 输入关键词时，列表可即时筛选。
3. 点击搜索按钮时，按当前关键词筛选。
4. 搜索不跨端类型，例如 APP端搜索只查 APP端广告位。
5. 搜索结果为空时，展示空状态：暂无匹配广告。
6. 搜索后分页总条数展示为当前命中的数量。
7. 点击重置后，清空搜索关键词、恢复列表和原总条数。

### 7.6 分页

分页展示：

| 字段 | 说明 |
| --- | --- |
| 总条数 | 当前端类型下广告位总数；搜索后展示命中数量。 |
| 每页条数 | 默认 20条/页。 |
| 页码 | 展示当前页和可切换页码。 |

本期原型仅展示分页样式；真实系统需要按接口返回数据分页。

### 7.7 底部按钮

| 按钮 | 规则 |
| --- | --- |
| 取消 | 关闭弹窗，不保存本次修改。 |
| 确定 | 保存当前模板的广告配置，并关闭弹窗。 |
| 关闭 X | 关闭弹窗，不保存本次修改。 |

## 8. 数据与接口建议

### 8.1 获取模板列表

请求参数：

| 参数 | 说明 |
| --- | --- |
| productType | 产品类型。 |
| templateName | 模板名称关键词。 |
| contentType | 内容类型。 |
| createStartTime | 创建开始时间。 |
| createEndTime | 创建结束时间。 |
| pageNum | 页码。 |
| pageSize | 每页条数。 |

返回字段：

| 字段 | 说明 |
| --- | --- |
| templateId | 模板ID。 |
| templateName | 模板名称。 |
| contentTypes | 内容类型列表。 |
| status | 状态。 |
| creator | 创建人。 |
| createTime | 创建时间。 |

### 8.2 获取模板广告配置

请求参数：

| 参数 | 说明 |
| --- | --- |
| templateId | 模板ID。 |
| terminal | 端类型：APP 或 WEB。 |
| keyword | 广告位名称关键词，可选。 |
| pageNum | 页码。 |
| pageSize | 每页条数。 |

返回字段：

| 字段 | 说明 |
| --- | --- |
| adId | 广告位ID。 |
| adName | 广告位名称。 |
| terminal | 端类型。 |
| checked | 当前模板是否已配置该广告位。 |
| disabled | 当前广告位是否不可配置。 |

### 8.3 保存模板广告配置

请求参数：

| 参数 | 说明 |
| --- | --- |
| templateId | 模板ID。 |
| terminal | 端类型。 |
| selectedAdIds | 当前端类型下勾选的广告位ID列表。 |

保存规则：

1. 保存时以后端接收到的 `selectedAdIds` 作为当前端类型最终配置。
2. 只保存当前端类型，不影响另一个端类型的广告配置。
3. 保存成功后写入操作日志。
4. 保存失败时保留弹窗，并展示错误提示。

## 9. 权限与日志

权限规则：

1. 只有具备产品模板管理权限的管理员可访问页面。
2. 只有具备广告配置权限的管理员可点击“配置广告”并保存。
3. 无权限时，按钮置灰或隐藏，后端必须二次校验。

操作日志：

| 日志字段 | 说明 |
| --- | --- |
| 操作对象 | 产品模板。 |
| 模板ID | 被操作模板ID。 |
| 操作类型 | 配置广告。 |
| 操作人 | 当前登录管理员。 |
| 操作时间 | 保存时间。 |
| 变更前 | 保存前广告位配置。 |
| 变更后 | 保存后广告位配置。 |

## 10. 验收标准

- 点击列表任意一行“配置广告”，打开配置广告弹窗。
- 弹窗标题、APP端/WEB端页签、搜索区、广告位列表、分页、取消和确定按钮展示正确。
- 默认选中 APP端，并展示 APP端广告位。
- 点击 WEB端后，广告位列表切换为 WEB端数据。
- 在搜索框输入关键词后，列表只展示当前端类型下命中的广告位。
- 点击搜索按钮后，筛选结果与输入关键词一致。
- 点击重置后，搜索框清空，列表恢复当前端类型完整数据。
- 搜索无结果时展示“暂无匹配广告”。
- 勾选或取消勾选广告位后，复选框状态正确变化。
- 点击取消或关闭 X 后，弹窗关闭且不保存修改。
- 点击确定后，保存当前模板广告配置并关闭弹窗。
- 保存成功后写入操作日志。
- 无广告配置权限的用户不能保存广告配置。
